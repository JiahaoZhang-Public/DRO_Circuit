## Goal

Determine whether the DRO advantage persists across different dataset sizes, or whether it diminishes with more examples (as scores become more stable).

## Environment

- **Server:** gpuhub-root-rtx4090-48
- **GPU:** RTX 4090 48GB
- **Model:** GPT-2 small
- **Task:** IOI, K=5 corruptions
- **N examples:** [50, 100, 200, 400]
- **Edge budgets:** [100, 200, 400]
- **Seed:** 42
- **Commit:** TBD

## Exact Commands

```bash
cd /root/projects/DRO_Circuit && conda activate dro

for N in 50 100 200 400; do
    echo "=== N=${N} ==="
    python experiments/comprehensive_experiment.py \
        --n_examples ${N} --batch_size 25 --device cuda --seed 42 \
        --edge_budgets 100 200 400 \
        --output_dir outputs/exp06_nscale_n${N}
done

# Compare across N
for N in 50 100 200 400; do
    echo "=== N=${N} ==="
    python -c "
import json
with open('outputs/exp06_nscale_n${N}/summary.json') as f:
    s = json.load(f)
for b in [100, 200, 400]:
    erm = s.get(f'dro_erm_mean_n{b}', {})
    dro = s.get(f'dro_max_n{b}', {})
    print(f'  n={b}: ERM worst={erm.get(\"worst\",\"?\"):.4f}, DRO worst={dro.get(\"worst\",\"?\"):.4f}')
"
done
```
